
# Expense Tracker - Starter Project

## Overview
Starter Spring Boot project for an Expense Tracker web app (Thymeleaf + MySQL).

## Setup
1. Ensure Java 25 and MySQL are installed.
2. Create database:
   ```sql
   CREATE DATABASE expense_tracker;
   ```
3. Update `src/main/resources/application.properties` .
4. Run:
   ```
   mvn spring-boot:run
   ```
5. Open http://localhost:8080
